🏙️ Cleveland Trip — Icon Generator

This page converts the bundled SVG into the PNG sizes iOS Safari needs for "Add to Home Screen." Run it once.

How to use:
  1. Click Generate & Download below.
  2. Your browser will download three PNG files: icon-192.png, icon-512.png, apple-touch-icon.png.
  3. Move them from your Downloads folder into this project folder (next to index.html).
  4. Commit and push. Done — you can delete this file afterwards.
